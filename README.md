# Clipping-online.com
The online business website equipped with file exchange / order placement system

## Technology utilized:
- Express
- EJS - view engine
- Flash - flash 
- NodeMailer
- Sequelize - for connecting the database
- Multer - for file transfer (to be replaced by BusBoy due to performance issues)
